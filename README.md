<p align="center">Create your very own unique pixel art piece.</p>

## The Problem

In recent months, NFT arts are getting more and more trending around the world. With my enthusiasm for cool technologies, I might as well create something that can help me create beautiful and collectible artwork. However, I'm not planning to dive too deep into the rabbithole of blockchain anytime soon, so this project will not contain any blockchain technology.

## The Solution

I've made a web app that let users create colorful and collectible artworks in a 7x7 grid and share them to the world.

## ⌨️ Setup

If you want to run the website on your local machine:
1. Clone the repository: `git clone https://github.com/melvinchia3636/colorpix`
2. Install all required dependencies using `npm install`
4. Run `npm run dev` and you're good to go.

## 📈 Status

This project is still under development, new features are being added constantly. If any bugs are found, please file an issue here, and I'll resolve it ASAP.
